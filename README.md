# 📊 Employee Performance Analysis Dashboard | Advanced Excel

## 📌 Project Overview

This project analyzes employee performance using Advanced Excel to identify high-performing employees, evaluate regional performance, and generate business insights through an interactive dashboard.

---

## 🎯 Project Objectives

- Analyze employee performance.
- Identify high-performing and low-performing employees.
- Evaluate regional performance.
- Build an interactive dashboard for management.
- Generate actionable business insights.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- IF Function
- Conditional Formatting
- Slicers

---

## 📂 Data Preparation

- Removed duplicate records.
- Handled missing values.
- Corrected inconsistent data.
- Created a new Performance Category using the IF function to classify employees as High Performer, Average Performer, and Low Performer.

---

## 📈 Excel Features Used

- Pivot Tables
- Pivot Charts
- IF Function
- Conditional Formatting
- Slicers
- Sorting & Filtering

---

## 📊 Dashboard Features

- Employee Performance Dashboard
- Regional Performance Analysis
- Performance Category Distribution
- Interactive Filters
- Dynamic Pivot Charts

---

## 💡 Business Insights

- Identified top-performing employees and regions.
- Highlighted low-performing employees requiring improvement.
- Compared regional performance for better decision-making.
- Simplified employee performance tracking through an interactive dashboard.

---

## ✅ Business Recommendations

- Recognize and reward top performers.
- Provide training for low-performing employees.
- Focus on improving performance in underperforming regions.
- Use dashboard insights for regular performance monitoring.
